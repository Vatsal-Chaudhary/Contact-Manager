# Contact Management System

This is a Contact Management System developed using Python, Tkinter for GUI, and MySQL for the database.

## Features

- Add Contact
- View Contacts
- Delete Contact
- Update Contact

## Prerequisites

- Python 3.x
- Tkinter
- MySQL
